## Project Overview

This is a Cocos Creator 3.8.8 idle game project. Cocos Creator is a game engine with an editor-based workflow. The game features an attribute/equipment/buff system for character progression.

## Architecture

### Component-Based Architecture
- Components extend `cc.Component` and use the `@ccclass` decorator
- The main entry point is `Main.ts` attached to the Canvas node in `Main.scene`
- Components access child nodes via `this.node.getChildByPath()` and components via `getComponent()`

### Attribute System (`AttributeManager`)
The core game logic centers around `Controller/AttributeManager.ts`:

- **Purpose**: Calculates final character attributes from base stats, equipment, and buffs
- **Formula**: `final = (base + flatSum) * (1 + percentSum)`
- **Key types** (defined in `types/types.ts`):
  - `AttrId`: "atk" | "def" | "hp" | "maxHp" | "critRate" | "speed"
  - `AttrModifier`: { attrId, type: "flat" | "percent", value }
  - `CharacterRuntime`: { baseAttrs, equipped, buffs }
  - `EquipmentTemplate`: Static equipment data with modifiers
  - `BuffInstance`: Temporary modifiers with `remainingMs`

- **Equipment slots**: "weapon", "armor", "helmet", "accessory"
- **Modifier sources tracked**: base, equipment, buff (each has `sourceType`, `sourceId`, `sourceName`)

## Development Workflow

### Cocos Creator Editor
- Open the project in Cocos Creator 3.8.8 for scene editing, asset management, and building
- The editor manages `library/` and `temp/` directories (gitignored)

### TypeScript
- Custom code goes in `assets/**/*.ts`
- `tsconfig.json` extends `./temp/tsconfig.cocos.json` (generated by Cocos Creator)
- Import Cocos modules: `import { _decorator, Component, ... } from 'cc'`
- Use decorators: `@ccclass('ClassName')`, `@property`

### File Structure
```
assets/
  Main.ts              # Canvas component (entry point)
  Main.scene           # Main scene file (JSON, editor-managed)
  Controller/          # Game logic managers
    AttributeManager.ts
  types/
    types.ts           # Game type definitions
```

## Important Notes

- Scene files (`.scene`) are JSON and should not be hand-edited; use the Cocos Creator editor
- Meta files (`.meta`) accompany assets for Cocos Creator's internal tracking
- Do not modify `library/`, `temp/`, `local/`, `profiles/` - these are editor-managed
